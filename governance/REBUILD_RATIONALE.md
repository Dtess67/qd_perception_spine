## Grounded Reasons for Rebuild

- Status: **Directly evidenced**
  - Issue: runtime behavior can vary through optional fallback paths in measurement logging.
  - Evidence basis: `resonance_metrics_logger.py` discovers optional v2 helpers and falls back to identity adjustment/local polarity when unavailable.
  - Rebuild pressure: constitutional traceability and cross-environment reproducibility require explicit mode disclosure.

- Status: **Directly evidenced**
  - Issue: contradiction-cooldown activation is externalized from the v2 kernel path.
  - Evidence basis: `resonance_kernel_v2.py` states contradiction detection/trigger is outside the module and must call `start_time_gate_cooldown_v0()`.
  - Rebuild pressure: safety-relevant HOLD/cooldown behavior needs explicit, testable end-to-end contracts, not optional caller discipline.

- Status: **Strongly inferred**
  - Issue: core processing is pre-shaped by semantic axis roles rather than a purely neutral lane.
  - Evidence basis: `axis_schema.py` defines conceptual axis meanings; `axis_roles.py` assigns influence/tendency semantics used by `resonance_kernel_v2.py`.
  - Rebuild pressure: current governing posture requires neutral-lane discipline and earned structure before semantic overlays.

- Status: **Strongly inferred**
  - Issue: governance model mismatch between QD_v3 and current Phase I constitutional obligations.
  - Evidence basis: `PROJECT_CONTRACT.md` locks QD_v3 to architecture/workflow optimization scope; `qd_perception_spine` governing artifacts require constitutional discoverability and inheritance mapping.
  - Rebuild pressure: the QD_v3 artifact set was strong for local workflow discipline, but it did not provide a single recoverable cross-session, cross-source inheritance map of rebuild causes, preserved contributions, and bounded gaps.

- Status: **Open reconstruction item**
  - Issue: missing constraint-bootstrap lineage prevents full continuity reconstruction.
  - Evidence basis: recursive search found no `constraint_bootstrap_v1*` under `X:\Dev\QD_Main\QD_v3`.
  - Rebuild pressure: unresolved constraint lineage must be bounded explicitly rather than carried as implicit memory.

- Status: **Strongly inferred**
  - Issue: local correctness in QD_v3 does not equal constitutionally governed cross-session continuity.
  - Evidence basis: `ARCHITECTURE.md` / `BUILD_NOTES.md` emphasize local loop completion; current governing artifacts define broader cold-session governance and inheritance duties.
  - Rebuild pressure: without constitutional closure, useful mechanisms remain vulnerable to drift and disconnected lineage.
